Lección 1: ¿Qué es un formulario en HTML?

Definición y Propósito

Un formulario en HTML es un componente fundamental de la web que permite a los usuarios enviar datos a un servidor. Piénsalo como un documento digital con espacios en blanco que el usuario puede rellenar.

Su propósito principal es la recolección de datos. Sin formularios, la web sería en gran medida un lugar de solo lectura. Gracias a ellos, podemos interactuar con las páginas, proporcionar información y recibir respuestas personalizadas.

Uso Común

Los formularios están por todas partes en la web. Algunos de los ejemplos más comunes incluyen:

  • Formularios de Inicio de Sesión (Login): Para autenticar usuarios y darles acceso a áreas restringidas.
  • Formularios de Registro: Para crear nuevas cuentas de usuario.
  • Formularios de Contacto: Para que los visitantes puedan enviar mensajes o preguntas.
  • Encuestas y Cuestionarios: Para recopilar opiniones o información específica.
  • Barras de Búsqueda: Un tipo simple de formulario para buscar contenido en un sitio.
  • Procesos de Compra (Checkout): Para introducir datos de envío y pago en tiendas online.

Reto

Piensa y anota tres sitios web que uses frecuentemente y que utilicen formularios. ¿Para qué los usan? Escribe tus respuestas en una lista a continuación, modificando este archivo HTML.

  • Ejemplo: En Facebook, uso el formulario de inicio de sesión para entrar a mi cuenta y el de "publicar" para compartir estados.
  • Google: Utilizo el formulario de búsqueda constantemente para encontrar información, y también los formularios de Gmail para escribir y enviar correos electrónicos.
  • Amazon: Uso formularios de búsqueda de productos, formularios de checkout para realizar compras, y formularios de reseñas para calificar productos que he comprado.
  • YouTube: Empleo el formulario de búsqueda para encontrar videos, el formulario de comentarios para interactuar con otros usuarios, y formularios para subir mis propios videos.
